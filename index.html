แก๊สและสมบัติของแก๊ส

เริ่มอ่านเนื้อหา ดูสรุป

1. บทนำ

สถานะแก๊สและสมบัติพื้นฐาน

สถานะแก๊ส (Gas State) คือ สถานะของสสารที่มีพลังงานสูง อนุภาคอยู่ห่างกันมาก และเคลื่อนที่ได้อย่างอิสระตลอดเวลา

📌 4 สมบัติเด่นของแก๊ส

1. รูปทรงไม่แน่นอน: ปริมาตรและรูปร่างจะเปลี่ยนไปตาม "ภาชนะที่บรรจุ" เสมอ
2. แรงยึดเหนี่ยวน้อยมาก: อนุภาคอยู่ห่างกัน จึงมีความหนาแน่นต่ำกว่าของแข็งและของเหลว
3. บีบอัดได้ง่าย: สามารถบีบอัด (กด) หรือลดปริมาตรลงได้มาก เพราะมีพื้นที่ว่างระหว่างโมเลกุลเยอะ
4. ฟุ้งกระจายรวดเร็ว: สามารถเคลื่อนที่ผสมกลมกลืนกับแก๊สอื่นได้อย่างรวดเร็ว (เช่น กลิ่นน้ำหอม)

🌡️ แบบจำลองการเคลื่อนที่ของโมเลกุลแก๊ส

2. ตัวแปรและหน่วยวัด

ตัวแปรสำคัญในสมบัติของแก๊ส

ปริมาตร (V)

1 L = 1,000 cm³ = 1,000 mL

1 L = 1,000 dm³

ความดัน (P)

1 atm = 760 mmHg = 760 torr

อุณหภูมิ (T)

การคำนวณเรื่องแก๊สต้องใช้อุณหภูมิสัมบูรณ์เป็นเคลวิน

K = °C + 273

จำนวนโมล (n)

แสดงถึงปริมาณสารของแก๊ส

n = g/M

ค่าคงที่ของแก๊ส (R)

ค่าคงที่สากลของแก๊ส ขึ้นกับหน่วยความดันและปริมาตร

R = 0.0821 L · atmmol · K

3. กฎของแก๊ส

ความสัมพันธ์ที่ต้องจำ

กฎของบอยล์ (Boyle's Law)

P₁V₁ = P₂V₂

ปริมาตรของแก๊สจะแปรผกผันกับความดัน เมื่ออุณหภูมิ (T) และจำนวนโมล (n) คงที่

กฎของชาร์ล (Charles's Law)

V₁/T₁ = V₂/T₂

ปริมาตรของแก๊สจะแปรผันตรงกับอุณหภูมิสัมบูรณ์ เมื่อความดัน (P) และจำนวนโมล (n) คงที่

กฎของเกย์-ลูสแซก

P₁/T₁ = P₂/T₂

ความดันของแก๊สจะแปรผันตรงกับอุณหภูมิสัมบูรณ์ เมื่อปริมาตร (V) และจำนวนโมล (n) คงที่

กฎของอาโวกาโดร

V₁/n₁ = V₂/n₂

ปริมาตรของแก๊สจะแปรผันตรงกับจำนวนโมล (n) เมื่ออุณหภูมิ (T) และความดัน (P) คงที่

ที่ STP แก๊สใดๆ 1 โมล จะมีปริมาตร 22.4 ลิตร

กฎรวมแก๊ส (Combined Law)

(P₁V₁)/T₁ = (P₂V₂)/T₂

เป็นการรวมกฎของบอยล์และกฎของชาร์ลเข้าด้วยกัน ใช้เมื่อสภาวะเปลี่ยนแปลงไปพร้อมๆ กัน

4. สมการแก๊สอุดมคติ

PV = nRT และการแตกสูตร

สมการหลัก

PV = nRT

ค่า R ที่ใช้บ่อยคือ 0.0821 L·atm/(mol·K) หรือ 8.314 J/(mol·K)

มวลและความหนาแน่น

n = g/M
M = gRT/PV
D = PM/RT

ความหนาแน่นของแก๊สแปรผันตรงกับความดัน และแปรผกผันกับอุณหภูมิ

5. หัวข้อเสริม

ความดันย่อยและการแพร่ของแก๊ส

กฎความดันย่อยของดอลตัน

Ptotal = P₁ + P₂ + P₃ + ...

ความดันรวมของแก๊สผสมที่ไม่มีปฏิกิริยาเคมีต่อกัน จะเท่ากับผลรวมของความดันย่อยของแก๊สแต่ละชนิดในส่วนผสมนั้น

เศษส่วนโมล (Mole Fraction)

X₁ = n₁/ntotal

อัตราส่วนจำนวนโมลของสารองค์ประกอบหนึ่งต่อจำนวนโมลทั้งหมด โดยความดันย่อยสัมพันธ์ตามสมการ: P₁ = X₁ · Ptotal

กฎการแพร่ผ่านของเกรแฮม

r₁/r₂ = M₂/M₁

อัตราการแพร่ผ่านของแก๊สจะเป็นสัดส่วนแปรผกผันกับรากที่สองของมวลโมเลกุลหรือความหนาแน่นของแก๊ส (แก๊สเบาแพร่เร็วกว่าแก๊สหนัก)
